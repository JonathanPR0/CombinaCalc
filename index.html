Arranjo simples:

Se você deseja saber a quantidade de maneiras que se pode agrupar n elementos distintos você faz uso do arranjo. Mas preste atenção! O arranjo só é usado quando a ordem dos elementos importa para o resultado final

Para obter o arranjo simples de n elementos tomados, p a p (p ≤ n), utiliza-se a seguinte expressão: An,p: n!/(n-p)!

Combinação simples:

As combinações são subconjuntos em que a ordem dos elementos não é importante, entretanto, são caracterizadas pela natureza dos mesmos.

Para calcular uma combinação simples de n elementos tomados p a p (p ≤ n), utiliza-se a seguinte expressão: Cn,p: n!/(p! * (n-p)!)
